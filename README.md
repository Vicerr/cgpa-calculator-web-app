# CGPA Calculator

A modern GPA/CGPA calculator built with [Next.js](https://nextjs.org/), [TypeScript](https://www.typescriptlang.org/), and [Tailwind CSS](https://tailwindcss.com/).

## Features

-   Add courses with course codes, loads, and grades
-   Organize courses per semester and per session
-   Automatically compute:
    -   GPA for each semester
    -   CGPA for each session (2 semesters per session)
    -   Overall CGPA across all sessions
-   Beautiful and responsive UI with Tailwind CSS

-   Node.js (v18 or later)
